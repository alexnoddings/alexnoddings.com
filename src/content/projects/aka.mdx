# Summary

`aka` is my personal link shortening/vanity service.

import AkaArchitecture from "$/content/projects/AkaArchitecture.astro";

<AkaArchitecture/>

## Main site
Users interact with the main site which runs on Cloudflare Serverless Workers.
It redirects users based on the path they are accessing.

## Admin site
Admins create/update links via a SvelteKit application which runs on Cloudflare Serverless Workers.
This is protected by Cloudflare One.

## Database
Both Worker applications use a Cloudflare D1 Serverless database for data storage.
