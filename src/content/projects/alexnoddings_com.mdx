# Summary

This website! Part résumé, part portfolio, part CV PDF generator.

It started off as a single, hand-rolled HTML page hosted on Heroku over 10 years ago, and has slowly evolved since.
Today, the site is developed using [Astro](https://astro.build/) and [Tailwind CSS](https://tailwindcss.com/),
and compiled to static HTML/CSS and served from Cloudflare Workers.
