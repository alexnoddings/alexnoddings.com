# Summary

`Elvet` is a bespoke, ergo¹, ortholinear², split³, wireless⁴, mech⁵, Colemak-inspired⁶ keyboard.

## A translation for non-keyboard-nerds
1. `ergo`: designed to be **ergo**nomic for prolonged usage
2. `ortholinear`: key are in straight columns, not staggered like on a usual keyboard, making them easier to navigate between
3. `split`: the keyboard is in two halves, making positioning more ergonomic
4. `wireless`: the keyboard is battery-powered and connects wirelessly
5. `mech`: the keyboard uses mechanical keyswitches, not membranes or chiclets like on a usual keyboard, which are nicer to type on
6. `Colemak-inspired`: the key layout is different to QWERTY, and involves less traveling between keys

## Details for keyboard nerds
- Each half is powered by a Nice! Nano v2
- The switches are (currently) Kailh Twilight Chocs (linear, 35gf) in hot-swap sockets
- It runs a build of ZMK
- The layout is inspired by Colemak, but customised for my dev work
- It has extra layers for:
  - QWERTY compatability to avoid rebinding keys in every video game
  - Function keys for F1-F12 and also F13-F24, a part of the [USB HID spec](https://www.usb.org/sites/default/files/hut1_7.pdf) which is almost never used by anything, making them _fantastic_ for macros and shortcuts in apps
  - Numpad and media playback controls
